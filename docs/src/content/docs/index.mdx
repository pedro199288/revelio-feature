## Features

<CardGrid stagger>
  <Card title="Framework agnostic" icon="star">
    It can be used with any framework or simply vanilla JS.
  </Card>
  <Card title="Rich API" icon="star">
    Allows you to customize the tour to your needs: flow control methods, user
    interactivity, lifecycle hooks, CSS custom classes etc.
  </Card>
  <Card title="Light and no dependencies" icon="star">
    Revelio feature is a light library with no dependencies, it's only 4.8kb
    gzipped.
  </Card>
  <Card title="Typescript" icon="star">
    Revelio feature has Typescript support, allowing type safety and better
    developer experience.
  </Card>
  <Card title="Auto scrolls to elements" icon="star">
    Automatically scrolls to elements, even when this elements are in nested
    scollable containers.
  </Card>
  <Card title="Supports stacking contexts nesting" icon="star">
    Others feature discovery alternatives doesn't work with [stacking
    contexts](https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_positioned_layout/Understanding_z-index/Stacking_context)
    nesting, Revelio feature allows you to overcome this problem with its rich
    API.
  </Card>
  <Card title="Free and open source" icon="star">
    Revelio will be always free and open source, we are open to contributions
    and feedback!
  </Card>
</CardGrid>

## Get started

Install the dependency and start building the feature tour your users deserve.

```bash
npm install revelio-feature
```

Create your first tour.

```js
import { Revelio } from 'revelio-feature';

const revelioTour = new Revelio({
  options: {
    // ... custom global options
  },
  journey: [
    {
      title: 'Welcome to my Amazing App!', // title of the step
      content: 'Click "Next" to get started discovering my app features.', // description of the step
      element: '#welcome-element', // CSS query selector
      options: {
        // ... custom options for this step
      },
    },
    // ... more steps
  ],
});
```
